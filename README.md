# FitCar - Premium Automotive Experience

[![Netlify Status](https://api.netlify.com/api/v1/badges/YOUR_NETLIFY_ID/deploy-status)](https://app.netlify.com/sites/fitcar-premium/deploys)
[![Vercel Deployment](https://img.shields.io/badge/Vercel-Deployment-blue?logo=vercel&logoColor=white)](https://vercel.com/import/git?repo=https://github.com/YOUR_GITHUB_USER/fitcar-premium)

FitCar is a premium, high-tech automotive platform designed for a seamless car buying and rental experience. Built with performance, accessibility, and modern design in mind.

## 🚀 Key Features

*   **Premium UI/UX**: Modern, responsive design with dark mode support.
*   **Dynamic Listings**: Advanced filtering and search for a vast inventory of vehicles.
*   **Car Comparison**: Side-by-side comparison of technical specifications.
*   **Wishlist System**: Save favorite cars for later.
*   **Interactive Dashboard**: Personalized user experience for managing bookings and profile.
*   **High Performance**: Lazy-loaded components and optimized assets for fast load times.

## 🛠️ Tech Stack

*   **Frontend**: React 18, Vite
*   **Styling**: Tailwind CSS
*   **Animations**: Framer Motion
*   **State Management**: Zustand
*   **Icons**: Lucide React
*   **Form Handling**: React Hook Form, Zod

## 📦 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/sagar/fitcar-premium.git
    cd fitcar-premium
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
